# ⚙️ Game Engine (Frontend)

## Descripción
Cliente del motor de juego. Renderizado sincronizado, input handling, predicción del cliente e interpolación.

## Módulos

### 1. ClientEngine.js
Integración con Three.js y socket.

### 2. Predictor.js
Predicción de movimiento del cliente.

### 3. Interpolator.js
Interpolación suave de movimientos remotos.
