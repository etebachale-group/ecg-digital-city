# 📦 Packet System (Frontend)

## Descripción
Cliente para manejar paquetes binarios del servidor. Parse, validación y state synchronization.

## Módulos

### 1. PacketClient.js
Cliente WebSocket para recibir/enviar paquetes.

### 2. StateSync.js
Sincronización de estado desde paquetes comprimidos.

### 3. PacketHandlers.js
Handlers específicos por tipo de paquete.
